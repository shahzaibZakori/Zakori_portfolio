# Shahzaib Zakori Portfolio

A modern React + Vite portfolio site built with Tailwind CSS and TypeScript. This project showcases the skills, projects, and experience of Shahzaib Zakori using a polished single-page layout and reusable React components.

---

## Features

- Built with **React 19** and **TypeScript**
- Powered by **Vite** for fast local development and production builds
- Styled using **Tailwind CSS** with a responsive, mobile-friendly layout
- Includes modular sections like **About**, **Archive**, **Contact**, **Deep Dive**, **Failures**, **Logs**, **Popup**, and **Terminal**

---

## Project Structure

- `src/main.tsx` - React app entry point
- `src/App.tsx` - Root application component
- `src/index.css` / `src/App.css` - Global and application styles
- `src/components/` - Reusable UI sections for the portfolio
- `public/` - Static assets served by Vite

---

## Getting Started

### Prerequisites

- Node.js (recommended latest LTS)
- npm

### Install dependencies

```bash
npm install
```

### Run in development mode

```bash
npm run dev
```

Open the local URL shown in the terminal to view the portfolio.

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

---

## Scripts

- `npm run dev` - Start the Vite development server
- `npm run build` - Build the project for production
- `npm run lint` - Run ESLint across the repository
- `npm run preview` - Preview the production build locally
